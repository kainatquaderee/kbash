# kbash
bash helpers and scripts for Kainat OS

# Bash Aliases Cheat Sheet

| Alias | Command | Description |
|-------|---------|-------------|
| ll    | ls -alF | List all files with details |
| la    | ls -A   | List all files except . and .. |
| l     | ls -CF  | List files in columns with indicators |
| gs    | git status | Show Git repository status |
| gp    | git push   | Push changes to remote repository |
| gd    | git diff   | Show changes not staged for commit |
| ga    | git add    | Add file(s) to staging |
| gc    | git commit | Commit staged changes |
| gco   | git checkout | Switch branches or restore files |
| ..    | cd ..       | Move up one directory |
| ...   | cd ../..    | Move up two directories |
| ....  | cd ../../.. | Move up three directories |
